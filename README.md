# terraform-ansible-simplewebsite

## How to Setup

### Terraform [main.tf]

1. Ensure AWS credentials are vaild

### Ansible [inventory]

1. Ensure Ansible is present on local machine

## How to Run

1. `$ terraform init`
2. `$ terraform plan` (optional)
3. `$ terraform apply`

### To test

`$ curl <public_ip_address>`
